# HOMEHARDWARE

Hardware Store Management System - A comprehensive PHP-based inventory and sales management solution.

## Features

- **User Management**: Role-based access control with admin panel
- **Inventory Management**: Product tracking, stock levels, barcode scanning
- **Sales Management**: Point-of-sale system with customer management
- **Purchase Orders**: Supplier management and purchase tracking
- **Reporting**: Analytics and comprehensive reporting dashboard
- **Admin Panel**: User management, role assignments, system settings

## Technology Stack

- **Backend**: PHP with custom MVC framework
- **Database**: MySQL
- **Frontend**: Bootstrap 4, jQuery, Font Awesome
- **Architecture**: Model-View-Controller (MVC) pattern

## Installation

1. Clone the repository
2. Configure database settings in `app/config.php`
3. Import database schema from `database/migrations/`
4. Set up web server to point to the project root
5. Access the application via web browser
