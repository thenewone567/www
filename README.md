# Hardware Shop Management System

This is a web-based application for managing a hardware shop. It includes features for managing inventory, sales, purchases, customers, suppliers, and more.

## Recent Changes

### Responsive Dashboard

A new responsive dashboard has been created using Bootstrap 5 and Chart.js. The dashboard provides a comprehensive overview of the shop's performance, including:

*   Sales overview (today, this week, this month)
*   Top-selling items
*   Pending restock
*   Low stock alerts
*   Quick actions
*   Charts for inventory and financials

The dashboard is integrated into the existing PHP application and is accessible from the main navigation.

### Navigation

The main navigation has been updated with a more detailed and organized structure. It includes nested dropdowns for various sections like Sales, Purchases, Inventory, and more, providing a better user experience.
